Endpoints Disponíveis na API

Recurso Método HTTP URL Descrição
Postagens GET /postagens Retorna todas as postagens
Postagens GET /postagens/:id Retorna a postagem pelo id
Postagens POST /postagens Cria uma postagem
Postagens DELETE /postagens/:id Remove uma postagem pelo id
Postagens PUT /postagens/:id Atualiza uma postagem pelo id
